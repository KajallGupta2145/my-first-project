# Power BI Mobile Sales Analysis Project

## Project Overview
This project focuses on analyzing mobile sales data using an interactive Power BI dashboard.

## Tools Used
- Power BI
- Excel (as data source)

## Key Insights
- Monthly and yearly sales trends
- Region-wise sales performance
- Product and brand-wise analysis
- Top-performing categories

## File Included
- Dashboard.pbix

## Objective
To visualize sales data and generate meaningful business insights using Power BI.
